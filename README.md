# Python Core Practice 

This repository contains structured Python programs developed to strengthen fundamental programming concepts and problem-solving skills.

## Programs Included

1. Operators Practice
   - Arithmetic, relational, logical, and assignment operators

2. Student Grade System
   - List of dictionaries
   - Total, average, grading logic
   - Topper identification

3. Secure Password Generator
   - Custom exception handling
   - random and string modules
   - Input validation

4. String Operations
   - Character iteration
   - Vowel counting
   - String reversal
   - Palindrome check
   - Case conversion

5. Four Sum Problem (DSA)
   - Unique quadruplets finder
   - Set usage for duplicate removal
   - Target-based combination logic

## Concepts Demonstrated

- Functions
- Custom Exceptions
- Built-in Modules
- Input Handling
- Nested Loops
- Dictionary and Set usage
- Problem-solving logic
